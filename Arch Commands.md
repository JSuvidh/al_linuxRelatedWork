Some useful Manjaro(Arch) commands
==================================

- Pycharm (community) 
  `pacman -S pycharm-community-edition`
- Vim
  `pacman -S vim`
- Virtual Environment
  `pacmman -S python-virtualenv`
  
    or
  `pacman -S python`    //this install python-3.3 venv
- Tree
  `pacman -S tree`
